Project Details

LearnMate — AI Assistant Platform

Abstract: LearnMate is a full-stack assistant focused on document Q&A, OCR-based ingestion, and workflow automation. Built to run locally with privacy and offline-first models.

Objectives

  • Provide interactive document Q&A with semantic search and citations
  • Support PDF, images, and slides ingestion via OCR
  • Secure multi-user authentication & role management

Architecture & Tech Stack

  • Backend: Flask (REST + WebSocket)
  • LLM: Ollama / local Phi-3 Mini for reasoning
  • OCR & Vision: Tesseract / TrOCR / Donut routing
  • Storage: SQLite + embeddings via FAISS/Chroma
  • Frontend: Vanilla JS + Tailwind / Terminal themed UI

Key Contributions

  • Designed prompt chaining for multi-step reasoning
  • Built PDF ingestion + OCR pipeline and structured outputs
  • Implemented secure auth and rate-limited endpoints
  • Stress-tested for injection and file parsing attacks
Code (public) Stack: Flask · Ollama · Tesseract · FAISS

CyberSense — Multi-Engine Antivirus Scanner

Abstract: CyberSense aggregates detections from multiple AV engines and heuristics to produce an explainable threat report and confidence scores.

Objectives

  • Automate scanning using multiple AV engines
  • Normalize findings and present consolidated report
  • Provide scoring & remediation suggestions

Architecture & Tech Stack

  • Orchestration: Python workers + Celery
  • Analysis: YARA signatures, static hashing, sandboxing stubs
  • Reporting: HTML & JSON reports with CVE/CWEs mapping

Key Contributions

  • Designed detection normalization layers
  • Implemented consensus scoring and false-positive suppression
  • Built UI for browsing reports and timelines
Code Stack: Python · Celery · Redis · YARA

Highlights

InfiWasTex — Autonomous Waste Detection Robot

Abstract: InfiWasTex is a robotics project combining cameras, YOLO detection, and SLAM navigation to autonomously detect and collect waste.

Key Systems

  • Perception: YOLOv8 / custom datasets
  • Navigation: LiDAR + Visual SLAM (RTAB-Map / ORB-SLAM variants)
  • Control: ROS2 stack and embedded microcontroller for gripper control

Contributions

  • End-to-end pipeline for detection → localization → pickup
  • Model training and dataset curation
  • Deployment on edge hardware (Jetson / Raspberry / NVidia)
Code Stack: ROS2 · YOLO · Jetson
